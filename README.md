# Decision Tree from Scratch

## Описание проекта
Реализация алгоритма решающего дерева классификации с нуля на Python с использованием только NumPy.

## Цель
Построение интерпретируемой модели классификации без использования готовых реализаций, включая ручную реализацию критерия разбиения и логики построения дерева.

## Реализованный функционал
- Полная реализация алгоритма Decision Tree Classifier
- Поддержка:
  - числовых (real) признаков
  - категориальных (categorical) признаков
- Критерий разбиения: индекс Джини
- Векторизованный поиск оптимального сплита (без циклов в вычислении критерия)
- Обработка edge-cases (константные признаки, пустые разбиения)
- Контроль переобучения:
  - max_depth
  - min_samples_split
  - min_samples_leaf

## Алгоритм построения
- Сортировка признаков и вычисление всех возможных порогов
- Оценка качества разбиения по критерию Джини
- Рекурсивное построение дерева
- Остановка по критериям:
  - чистота узла
  - ограничения глубины
  - минимальный размер подвыборки

## Особенности реализации
- Векторизованный расчёт критерия Джини через NumPy
- Оптимизированный поиск лучшего сплита
- Поддержка категориальных признаков через упорядочивание по target-rate
- Рекурсивная структура дерева в виде словаря

## Технологии
- Python
- NumPy
- collections.Counter

## Результат
Реализован полностью кастомный Decision Tree Classifier, с поддержкой числовых и категориальных признаков и контролем переобучения.
